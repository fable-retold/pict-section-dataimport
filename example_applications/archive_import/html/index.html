Archive.org Import

Generate Archive.org-shaped comprehensions from a CSV — a complex, non-Meadow schema defined entirely in app config. Each item fans out into an ArchiveItem, its ArchiveCollections, and the item↔collection joins (the shape of an archive.org item, like the General Mills Monster Cereals commercials). No live server: generate, preview, and download the comprehension JSON.

Sample CSV: archive-items-sample.csv
